## 🚀 Projects

### 🔐 Authentication System — Enterprise-Grade Auth Platform
> `Java` `Spring Boot` `Spring Security` `JWT` `BCrypt` `Bucket4j` `React` `TypeScript` `Tailwind` `MySQL`

A complete full-stack authentication platform built to production security standards — not a boilerplate, but a system designed from scratch with real-world threat modelling.

**Security highlights:**
- SHA-256 hashed tokens · BCrypt strength-12 password storage · OTP hashed before DB storage
- `TransactionSynchronization.afterCommit()` — emails only fire on successful DB commit, never on rollback
- Account lockdown → OTP-gated reactivation → freeze on exhaustion → support-gated recovery chain
- Email enumeration prevention · per-email rate limiting via Bucket4j · Geo-IP anomaly detection
- JWT interceptors on frontend for silent token refresh and automatic Authorization header injection
